# Projeto: Calculadora Inteligente em Java 🚀

### 📝 Descrição
Este projeto consiste em uma calculadora interativa desenvolvida em Java que realiza operações matemáticas fundamentais. O foco principal foi a organização do código e a implementação de uma interface amigável via console para o usuário.

### ⚙️ Funcionalidades
*   Operações Básicas: Soma, Subtração, Multiplicação e Divisão.
*   Tratamento de Erros: Verificação de divisão por zero.
*   Loop de Repetição: O usuário pode realizar múltiplos cálculos sem precisar reiniciar o programa.

### 🚀 Tecnologias e Conceitos
*   **Linguagem:** Java (JDK 17).
*   **Entrada de Dados:** Classe `Scanner`.
*   **Estruturas de Controle:** `switch-case` para seleção de operações e `do-while` para o menu interativo.

### 🔍 Visão de QA (Quality Assurance)
Como estudante focado em QA, este projeto foi utilizado para praticar cenários de teste, como:
1.  **Cenário Positivo:** Verificação de cálculos com números inteiros e decimais.
2.  **Cenário Negativo:** Como o sistema se comporta ao tentar dividir um número por zero.
3.  **Validação de Entrada:** Teste de comportamento ao inserir caracteres inválidos no menu.
