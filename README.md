Repositório dedicado ao armazenamento e organização dos meus estudos de programação. O foco aqui é o domínio da linguagem Java e a aplicação de boas práticas de Quality Assurance.

*   **Estruturas de Controle:** Lógica de decisão (if/else, switch) e repetição.

*   **Stack:** Java 21 + Spring Boot + Maven.
*   **Ferramentas:** VS Code e Git Bash (Windows).
*   **Foco em QA:** Validação de entradas e organização de caminhos (paths) para automação.
