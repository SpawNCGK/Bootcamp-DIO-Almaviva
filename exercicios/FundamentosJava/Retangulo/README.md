# Exercício: Área do Retângulo 📏

### 📝 Descrição
Cálculo de área e perímetro de um retângulo, reforçando o uso de múltiplas entradas de dados (base e altura).

### 🔍 Visão de QA
Este tipo de programa é ideal para praticar **Testes de Unidade**, verificando se o cálculo permanece correto mesmo com números decimais ou valores negativos.
