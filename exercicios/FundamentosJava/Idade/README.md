# Exercício: Verificador de Idade 🔍

### 📝 Descrição
Este exercício foca na lógica de cálculo de idade a partir do ano de nascimento, preparando a base para estruturas condicionais futuras.

### 🛠️ Como rodar
1. Tenha o JDK instalado.
2. No terminal, use: `javac Idade.java`
3. Execute: `java Idade`
