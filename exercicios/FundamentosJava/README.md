# Módulo: Fundamentos de Java 📚

### 📝 Sobre
Este diretório reúne os primeiros desafios e exercícios práticos desenvolvidos durante o início do meu treinamento em Java. Aqui foquei em dominar a sintaxe básica, manipulação de variáveis e interação com o usuário via console.

### 🗂️ Exercícios Incluídos
*   **Diferença de Idade:** Prática de cálculos aritméticos simples.
*   **Idade:** Lógica de entrada e processamento de dados.
*   **Quadrado & Retângulo:** Exercícios de geometria para fixação de operadores matemáticos e tipos primitivos.

### 🛠️ Competências Desenvolvidas
*   Uso da Classe `Scanner` para leitura de dados.
*   Declaração e inicialização de variáveis (`int`, `double`, `String`).
*   Organização de código limpo (Clean Code).
*   Versionamento de código com Git/GitHub.
