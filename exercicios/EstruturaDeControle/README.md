# Módulo: Estruturas de Controle em Java 🚀

## 📝 Sobre

Este diretório reúne os exercícios práticos focados no domínio de fluxos de repetição e condicionais lógicas. Aqui o objetivo foi avançar além da sintaxe básica, criando programas que tomam decisões complexas e processam dados de forma dinâmica.

## 📁 Exercícios Incluídos

* **Tabuada:** Automação de cálculos matemáticos com laços de repetição.
* **Cálculo de IMC:** Implementação de lógica condicional para classificação de saúde.
* **Ordem Decrescente:** Manipulação de intervalos e filtros de paridade (Par/Ímpar).
* **Verificador de Resto:** Controle de fluxo avançado com condições de parada dinâmica.

## 🛠️ Competências Desenvolvidas

* Uso de estruturas de repetição (`for`, `while`).
* Controle de fluxo com as palavras-chave `break` e `continue`.
* Aplicação de operadores lógicos e aritméticos (Módulo `%`).
* **Mindset de QA:** Validação de entradas e tratamento de regras de negócio.
